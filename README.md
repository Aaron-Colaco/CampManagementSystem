<h1>Camp Management Full-Stack Web Application</h1>
<h2>Avondale College Taurewa Camp Gear Hire System</h2>

<p>
This project was developed as a real world solution for Avondale College’s Taurewa Camp.  
I volunteered to work with the camp staff to independently design and build a system to replace their manual gear hire process.
</p>

<hr>

<h3>Features</h3>
<ul>
  <li>Automates key tasks such as invoice generation and gear allocation</li>
  <li>Provides staff with an interface to easily track gear hires and returns</li>
  <li>Integrates with the school’s SQL Server and web server, ensuring reliable data storage and secure access across the school network</li>
</ul>
